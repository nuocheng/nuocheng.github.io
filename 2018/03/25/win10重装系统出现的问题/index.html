win10重装系统出现的问题

发表于 2018-03-25 | 分类于 电脑

bitlocker加密后再次打开显示无法访问

windows+r 输入services.msc 打开本地服务页面

在本地服务里找到bitlocker drive encryption service然后双击打开属性，将启动类型改为自动，然后点击确定

重启一下服务

安装win10出现Win10提示“无法创建新的分区也找不到现有的分区”怎么办

1.shift + F10 出现命令提示符;
2.输入diskpart 开始启动程序：
3.输入 list disk 来检查自己的硬盘，防止更改硬盘错误
4.select disk 0 选择磁盘0；
5.clean 进行清除分区(会删除硬盘所有数据)
6.convert GPT 进行格式的转换；

开始启动的时候删除多余未安装成功的系统：

1、开始->控制面板->系统->高级->启动和故障恢复->设置
2、启动和故障恢复->默认操作系统->点开右边的下拉箭头->点击你想要的系统作为默认系统->确认；
3、或者打开编辑之后，删除相应系统的启动语句；
