java之Android基础

发表于 2018-05-05 | 分类于 Java

View视图的几个常用属性：

（1）、android:layout_width=”match_parent”以及”fill_parent”

这里的match_parent是如果有子视图，表示当前视图的宽度刚好能满足他的子视图的宽，如果没有则是刚好显示其中文本的宽度。

（2）、androida:layout_height=”match_parent”以及”fill_parent”

（3）、android:background=”#555555”

如果是一图片作为背景android:background="@drawable/dog"  这里的dog是图片的名字

（4）、android:layout_gravity

在父视图中的对齐方式其中有top left bottom right center 以及这些值的组合值 left|center  left|bottom

（5）、android:padding

作用：作用是设置视图的边距。android:paddingBottom="" android:paddingLeft=""  android:paddingRight=""

（6）、android:id

作用是确定视图的ID标识。android:id="@ + id/ID值"

（7）、android:alpha

一个视图可以设置透明度

TextView 视图

TextView视图特点是显示文本，但不允许用户编辑他上面的文本，其中作用相当如标签的作用
属性：

（1）、android:autoLink=”all,web,phone,email,map,none”

自动识别连接。当TextView视图上的文字办好web网址，email地址，phone号码等就将这部分特殊的文本显示为可点击的连接；

（2）、android:textColor=””TextView视图上的文字

（3）、android:textColorLink=””TextView视图上连接的颜色

（4）、android:textSize=””

（5）、android:textStyle=”normal ,bold, italic”

（6）、android:font-family=””

（7）、android:text=”” TextView视图上显示的字符串

（8）、android:gravity=”” 对齐方式

EditText视图

视图的常用属性

（1）、android:digits=”abad”

设置EditText视图只可以输入属性值中的字符，例如输入的文本只可以有a，b，c字符

（2）、android:inputType=”tiem,number,data,datatime”

设置视图中可以输入的数据类型，当设置成data时可以输入数字和“/”
当设置成是time的时候开头是输入数字和‘：’

（3）、android:singleLine=”true,false”

设置视图中是否是单行模式，默认是false

（4）、android:passworld=”true,false”

设置TextView视图作为密码视图

（5）、android:phoneNumber=”true,false”

设置TextView视图只可以输入电话号码

（6）、android:lines=”7”

设置TextView视图上显示的文本的行数

（7）、android:maxLines=”20”

设置TextView视图至多显示的文本的行数

（8）、android:minLinese=”8”

设置TextView视图上至少显示的文本的行数

控制插件位置的相关代码：tools:layout_editor_absoluteX=”69dp”

tools:layout_editor_absoluteY="189dp"

Button事件：

在程序代码中注册监视器，处理click事件
步骤：

(1) 确定button视图。首先给出需要给出click事件的button视图例如：

Button button=(Button)findViewById(R.id.my_button_add);

(2) 确定监视器。需要使用实现view OnClickListener 接口的类的对象作为Button视图的监视器，button视图通过如下方法注册监视器：

public void setOnClickListener(View.OnClickListener listener);

(3) 监视器重写View.OnClickListener 接口中的onClick(View v)方法：

例如：
    button.setOnClickListener(new View.OnClickListener)(){
        public void onClick(View v){
            //perform action on click;
        }
    });

当button视图上触发click事件后，监视器调用onclick(View view)方法，该方法中的view存放着当前触发click事件的button视图的引用


在视图资源文件中约定click事件
可以在button的视图资源文件中设置android:onClick
<Button    
    android:text="button"
    android:onClick="handle"<!--约定需要执行handle方法-->
>

定义约定的方法
public void handle(View view){
    //根据视图的约定所定义的handle方法
}
