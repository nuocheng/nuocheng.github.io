爬虫之表单交互与模拟登陆

发表于 2018-02-27 | 分类于 Python3

登录表达后的信息（实现表单交互requests的post方法）：

import requests

url=”https://www.douban.com/accounts/login“

params={
‘source’:’index_nav’,
‘form_email’:’‘,
‘form_password’:’*‘
}
html=requests.post(url,params)

print(html.text)

在豆瓣网上使用cookies直接进入：

import requests
url=’https://www.douban.com‘
headers={
‘Cookies’:’*‘

}

html=requests.get(url,headers=headers)

print(html.text)
