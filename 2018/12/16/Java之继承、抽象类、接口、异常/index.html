接口（用于解决单继承的方式）

1、接口的声明(interface 只写方法，不写方法体)：
public interface A{
public void show();
public abstract void show2();
}
2、接口的使用(implements)：
public class B implements A{
@Override
public void show(){
System.out.println(“这里是重写A类的方法体”);
}
@Override
public void show2(){
System.out.println(“这里是重写A类的方法体”);
}
}
