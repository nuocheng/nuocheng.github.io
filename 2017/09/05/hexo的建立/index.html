hexo的建立

发表于 2017-09-05

制作静态博客的步骤及代码
1、安装git
2、安装node.js
3、安装hexo

3、安装hexo的步骤：
使用git shell中输入 cd /
npm install hexo-cli -g

4、安装博客的时候需要准备的文件
cd /
hexo init hexo (这里是添加的一个文件，存放的是构建博客所要用到的所有的文件,这里的hexo是新建的文件名称）
cd hexo (然后进入这个文件下）
npm intal （安装依赖文件）
hexo g （编译 hexo generate）
hexo s （开启本地服务hexo sever）

5、 步数博客到服务器上
在git shell 上按Ctrl+c
在hexo（新建立的文件下找到_config.yml文件
deploy：
type：github
repo: https://github.com/yourname/blog.git
回到git shell输入命令：
hexo clean (清除public，当source文件夹中的部分字眼更改过后，特别是对文件进行了删除或者路径的改变之后，需要执行这个命令，然后重新编译）
hexo g （编译，编译后，会出现一个public文件夹，将所有的md文件编译成html文件）
hexo s （开启本地服务）
hexo d （部署博客到GitHub上，如果顺利，你就通过访问usename.gitub.io访问你的博客了）
