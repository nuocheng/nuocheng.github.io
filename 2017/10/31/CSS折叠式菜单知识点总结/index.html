CSS折叠式菜单知识点总结

发表于 2017-10-31 | 分类于 HTML

今天学了好多的东西：
在学折叠式菜单的时候学到的东西

target的使用方面

_blank :在新窗口中打开被连接文档
_self :默认，在相同的框架中打开被连接的文档
_parent :正在父框架集中带来被连接文档
_top :在整个窗口中打开被连接文档
framename:在指定的框架中打开被连接文档

details与summary的联合使用

summary与details标签一起使用。标题（summary）是可见的
当用户点击标题时会显示出详细信息

summary::-webkit-details-marker
是把显示的图标进行覆盖

:last-child选择器

例如：
    p:last-child{background:red;}
表示的是指定属于其父元素的最后一个子元素的p元素的背景色
